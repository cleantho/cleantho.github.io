    PROJETOS

    Onde teoria encontra prática

    Os projetos apresentados aqui refletem minha jornada de aprendizado e minha busca constante por domínio técnico. São aplicações que exploram diferentes arquiteturas, padrões de projeto, integrações entre serviços e construção de interfaces modernas. Cada projeto é uma oportunidade de experimentar, validar ideias e aprimorar minha forma de pensar e desenvolver software.

    dio-desafio-patterns

    O desafio envolveu a criação de um backend com utilização de design patterns.

    Neste projeto foi utilizado os padrões Singleton, Observer e Factory Method.

    No backend foi utilizado a linguagem Java com String Boot (spring-boot-starter-web, spring-boot-starter-data-jpa e spring-restdocs-mockmvc). Juntamente com banco de dados H2.

    No frontend foi utilizado o Vue + Vite.

    knowledge

    Neste projeto foi desenvolvido um sistema de divulgação de artigos para uma comunidade fechada.

    No backend foi utilizado a linguagem Javascript com servidor Express. Juntamente com dois bancos de dados: Postgres e Mongo.

    No frontend foi utilizado o Vue.

    Booker

    Este projeto simula a API Java de uma reserva de salas de reunião, utilizando o framework Spring Boot. Foi disponibilizado na nuvem da plataforma Railway.

    Devido incompatibilidade de versão do Java 23 no Railway, a versão no modo produção foi ajustada para Java 17.

    * Posteriormente a API foi desativada no Railway, para evitar custos.

    * Os projetos divulgados no Github são projetos criados no processo de aprendizado.
