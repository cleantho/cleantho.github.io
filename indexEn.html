dio-desafio-patterns

The challenge involved creating a backend using design patterns.

In this project, the Singleton, Observer and Factory Method patterns were used.

For the backend, the Java language was used with String Boot (spring-boot-starter-web, spring-boot-starter-data-jpa e spring-restdocs-mockmvc). Together with database H2.

For the frontend, Vue + Vite was used.

knowledge

In this project was developed a system for publishing articles to a closed community.

For the backend, the JavaScript language was used with the Express server. Together with two databases: Postgres and Mongo.

For the frontend, Vue was used.

Booker

This project simulates the Java API for a meeting room booking system, using the Spring Boot framework. It was deployed on the Railway platform.

Due to version incompatibility of Java 23 on Railway, the production version was adjusted to Java 17.

* Subsequently, the API was deactivated on Railway to avoid costs.
